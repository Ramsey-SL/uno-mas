# Premium Cocktail Lounge

## Description

Dark, moody, tequila-forward style for The Mezzanine and evening cocktail content. Cool older sibling voice — minimal, intentional, atmospheric. This is the only style that uses the Mezzanine sub-brand palette.

## Hidden prompt

Create a dark, sophisticated design for The Mezzanine — Uno Más's upstairs speakeasy and private event space. Deep black backgrounds, electric pink and magenta accents, moody single-light photography of cocktails or tequila. Refined but warm. Editorial, cinematic. Watercolor backgrounds welcome. The Mezzanine is the cool older sibling — never use main Uno Más casual energy.

## Visual keywords

moody, sophisticated, low-light, cinematic, agave-forward, refined, evening, tequila, speakeasy

## Color notes

⚠️ This style uses the **Mezzanine sub-brand palette** (NOT the main Uno Más palette):

- Background: Black `#000000` / Charcoal `#333333`
- Primary: Electric Pink `#E22790`
- Secondary: Magenta `#BF28BF`
- Tertiary: Ultra Violet `#93009B`
- Text on dark: White `#FFFFFF`

Black foundation. Electric Pink the only warm element — use intentionally. Watercolor backgrounds (Pink → Magenta → Ultra Violet gradient) are an approved Mezzanine design element.

## Typography notes

Per Ramsey's Mezzanine sub-brand:
- Titles: DIN Condensed VF (Demi Bold / Regular / Light)
- Body: Poppins (Black / Bold / Regular / Medium / Light / Thin)
- Accent callouts: Baka Too (decorative)

NEVER use Antonio (main Uno Más font) in Mezzanine designs.

## Layout notes

Generous negative space. Single hero element (cocktail glass, agave plant, leather lounge, fireplace). Headline placed deliberately, not in a colored block. Information hierarchy clean — name, date, time, price (when applicable), inquiry link.

## Photography notes

Single-light moody photography. 3/4 angle on cocktails showing condensation, pour, garnish. Dark Mezzanine interior — leather lounges, fireplace, dramatic lighting. Slight rim light. NO bright daylight. NO group shots. NO smiling faces.

## Example use cases

- Mezzanine private event marketing
- Tequila tastings
- Cocktail menu launches
- Espresso Margarita (Indaba collab) promos
- Evening reservation pushes
- Speakeasy / late-night content
- Anniversary / milestone announcements

## When NOT to use this style

- Lunch / daytime promotions → use Modern Mexican Minimal or Bold Cantina
- Family-friendly events → use Family Fiesta
- Cantina floor / general restaurant content → use Bold Cantina or Modern Mexican Minimal
- Anything food-forward (this is for atmosphere/cocktail/event) — use Modern Mexican Minimal for plate-led content

## Brand separation rule

Per Ramsey: NEVER mix Uno Más main palette (`#E22690`, Antonio font) with Mezzanine elements in the same design. They are separate visual identities.

## References

- NOT bright/colorful — dark is correct
