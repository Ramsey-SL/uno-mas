## Templates (`_TEMPLATES/`) — Code & Design Layouts

Reusable HTML and design templates organized by channel. Start here when building anything new.

| Folder | Subfolders | What Goes Here |
|--------|-----------|----------------|
| `Website/` | Pages, Landing_Pages, Components, Mockups | `.html` files for full pages, campaign LPs, reusable sections |
| `Email/` | Campaigns, Flows_Automations, Components | `.html` Klaviyo templates for one-off sends and automated flows |
| `Social_Media/` | Instagram_Feed, Instagram_Stories, Facebook | `.png`, `.canva`, `.psd` design templates |

**Naming convention:** `[channel]_[type]_[description]_v[version].[ext]`
Example: `web_page_homepage_v1.html` · `email_campaign_weekly-special_v1.html` · `social_feed_food-promo_v1.png`

See `_TEMPLATES/README.md` for full workflow instructions.

## Campaigns Structure (`08_CAMPAIGNS/`)

Each campaign gets its own subfolder:
```
08_CAMPAIGNS/
└── [campaign-name-year]/
    ├── Step 1 - briefs/       → strategy brief, creative brief, channel plan
    ├── Step 2 - Copy/         → all written copy (ads, email, SMS, social)
    └── Step 3 - creative/     → design files, mockups, exported assets
```

## When in Doubt — Where to Put It

| Type of file | Put it here |
|-------------|-------------|
| New photo | `_INBOX/` then sort to `02_PHOTO_LIBRARY/UM - Photos - By Category/[category]` |
| New video | `_INBOX/` then sort to `03_VIDEO_LIBRARY/UM - Videos - By Category/[category]` |
| New logo variant | `01_Brand_Assets/Logos/UnoMas/` or `Mezzanine/` |
| Campaign assets | `08_CAMPAIGNS/[campaign-name]/` |
| Print/signage file | `09_PRINT_MERCH/` |
| Event-specific content | `06_EVENTS/` |
| Old/outdated file | `_ARCHIVE/` |
| Not sure | `_INBOX/` |
