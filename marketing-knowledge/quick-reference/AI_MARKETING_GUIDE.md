# Uno Más — AI Marketing Guide
*Copy-paste prompts and workflows for every marketing task*

---

## How to Start Every AI Session

Always open with this context block — paste it at the top of any new conversation:

```
You are a marketing assistant for Uno Más Tacos & Tequila, a locally owned taco shop and bar in Spokane, WA with three locations (Wonder Building downtown, South Valley, and Monroe Street). The brand is bold, fun, community-driven, and unapologetically vibrant. Not corporate, not a chain — the friend who always knows where to eat. We also operate The Mezzanine, a private event venue above the Wonder Building location.

Brand voice: casual and confident, warm without being corny, witty without trying too hard. We use "Más" and "Uno Más" naturally. We don't use hashtags in body copy. We don't write like a press release. Write like a real person who loves tacos and their community.

Keep responses concise and punchy. When writing copy, give 2-3 options unless told otherwise.
```

## Social Media Prompts

### Instagram Caption (Food Post)
```
Write 3 Instagram captions for a photo of [DISH NAME].
Tone: fun, casual, Spokane-local energy.
Max 2 sentences + call to action.
No hashtags in the caption body.
Include one option that's minimal (under 10 words) and one that's more expressive.
```

### Instagram Caption (Event/Promo)
```
Write 3 Instagram captions announcing [EVENT/PROMO DETAILS].
Tone: excited but not hype-y, community feel.
Include the key info: date, time, location if relevant.
End with a clear CTA (link in bio, tag a friend, DM us, etc.)
```

### Instagram Stories (Engagement)
```
Write 5 Instagram Story poll/question prompts for Uno Más.
Topics: tacos vs burritos debate, favorite margarita flavor, weekend plans, etc.
Keep them quick, fun, very interactive.
```

### TikTok / Reel Hook Lines
```
Give me 10 hook lines for a TikTok/Reel video about [TOPIC].
The hook should stop the scroll in the first 2 seconds.
Format: one sentence, bold claim or question. Uno Más voice.
```

### Facebook Post
```
Write a Facebook post for [TOPIC/PROMO].
Audience: slightly older than Instagram, local Spokane community, families and regulars.
Tone: friendly neighborhood restaurant, not corporate.
Include key details, a warm CTA, and keep it under 150 words.
```

## Email & SMS Prompts

### Promotional Email
```
Write a Klaviyo promotional email for Uno Más about [OFFER/EVENT].
Subject line: 3 options (one curiosity, one direct, one emoji-forward)
Preview text: 1 option per subject line
Body: short and punchy — 3-4 sentences max before CTA
CTA button text: 2 options
Tone: warm, personal, like a text from a friend who runs a taco shop
```

### Welcome Email
```
Write a welcome email for new Uno Más email subscribers.
This is their first email after signing up — make them feel like they joined something real.
Include: a warm welcome, what to expect (deals, events, new menu items), a first-visit offer if applicable.
Keep it under 200 words. Tone: genuine, excited, not salesy.
```

### SMS Campaign
```
Write 3 SMS messages for [OFFER/EVENT].
Max 160 characters each.
Include: the deal/event, urgency/timeframe, a short link placeholder [LINK]
Tone: casual text from a friend, not a brand blast
Options: one playful, one straightforward, one FOMO-driven
```

### Re-engagement Email
```
Write a re-engagement email for Uno Más subscribers who haven't opened in 60+ days.
Subject line: 3 options — make them feel missed, not guilted
Body: brief, genuine, maybe a little self-aware about the silence
Offer: [INSERT OFFER] or leave open-ended
CTA: come back in, order online, or check out what's new
```

## Ad Copy Prompts

### Meta Ad (Awareness)
```
Write Meta ad copy for Uno Más targeting [AUDIENCE DESCRIPTION] in Spokane.
Goal: brand awareness / reach
Primary text: 3 options (under 125 characters)
Headline: 3 options (under 40 characters)
Description: 2 options (under 30 characters)
Angle: [choose — local pride / food porn / social occasion / value]
```

### Meta Ad (Conversion / Order Now)
```
Write Meta ad copy driving online orders for Uno Más.
Goal: click to order, direct response
Primary text: 3 options — lead with appetite, end with urgency
Headline: 3 options — feature the offer or the dish
CTA: "Order Now"
Tone: mouth-watering, fast, no fluff
```

### Google Ad
```
Write Google search ads for Uno Más targeting the keyword "[KEYWORD]".
Format: 3 headlines (max 30 chars each) + 2 descriptions (max 90 chars each)
Focus on: location (Spokane), what makes us different, CTA
Write 2 full ad variations
```

## Content Planning Prompts

### Monthly Content Calendar
```
Create a 4-week Instagram content calendar for Uno Más for [MONTH].
Include: 3-4 posts per week across food, drinks, events, community, and behind-the-scenes.
For each post: day, content type (photo/reel/story), topic, brief caption direction.
Note any upcoming holidays, local events, or seasonal hooks for [MONTH].
Format as a simple table.
```

### Weekly Content Brief
```
Give me a weekly content brief for Uno Más for the week of [DATE].
Include: 3 Instagram posts, 5 Stories, 1 TikTok/Reel idea, 1 Facebook post.
Hook into: [any current promo, event, or seasonal moment]
For each: platform, format, topic, angle, and one-line caption direction.
```

### Campaign Concept
```
Brainstorm 5 campaign concepts for Uno Más around [THEME/OCCASION/GOAL].
For each concept:
- Campaign name
- Core idea (1-2 sentences)
- Key channels (social, email, SMS, in-store)
- Hero content idea
- Offer or hook
Keep ideas Spokane-grounded and on-brand — fun, local, real.
```

## Review Response Prompts

### Positive Review Response
```
Write a response to this 5-star Google/Yelp review for Uno Más:
"[PASTE REVIEW]"
Tone: grateful, personal, not scripted. Mention something specific from their review.
Max 3-4 sentences. Sign off casually — no "Sincerely, The Uno Más Team."
```

### Negative Review Response
```
Write a response to this negative review for Uno Más:
"[PASTE REVIEW]"
Tone: calm, empathetic, no defensiveness. Acknowledge the issue, offer to make it right.
Don't make excuses. Keep it short — 3-4 sentences max.
End with a direct way to reach us (email or phone).
```

## Catering / Events Prompts

### Catering Pitch Copy
```
Write a short pitch for Uno Más catering targeting [AUDIENCE: offices / weddings / birthday parties / corporate events].
Format: 3-4 sentences + bullet list of what's included
Tone: appetizing, easy, stress-free
CTA: get a quote, call us, visit the website
```

### Mezzanine Event Copy
```
Write event venue copy for The Mezzanine at Uno Más targeting [AUDIENCE: corporate events / private parties / weddings / birthdays].
Highlight: atmosphere, capacity, food & drink from Uno Más, private/semi-private options.
Tone: elevated but approachable — this isn't a stuffy venue, it's a great time with great food.
Length: 2-3 short paragraphs + CTA
```

## Useful Modifiers

Add any of these to the end of any prompt to adjust the output:

- `Make it shorter.` / `Cut it in half.`
- `Make it more casual / more elevated / more local-feeling.`
- `Add a sense of urgency.`
- `Write it like a text message.`
- `Remove any clichés or corporate-sounding phrases.`
- `Don't use the word "delicious", "amazing", "perfect", or "experience".`
- `Give me a version for each location: Wonder Building, South Valley, Monroe.`
- `Make a version for Spanish-speaking audiences.`
