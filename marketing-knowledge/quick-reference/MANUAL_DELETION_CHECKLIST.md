## How to Delete Empty Folders in Finder

1. Open Finder → navigate to `Uno_Mas_HQ / Uno Mas Marketing HQ /`
2. Right-click any folder below → **Move to Trash**
3. If a folder appears to have contents, open it first — it may contain only `.DS_Store` (safe to delete the whole folder)
4. When done, empty the Trash
